# neos-gallery
A Neos CMS package with a gallery node type for NEOS 3.0

## Packages
- Swipebox 1.4.4

## Installation
Add the package in your site package composer.json

```
"require": {
     "obisconcept/neos-gallery": "~2.0.0"
 }
 ```

## Requirements
The package requires jQuery which is not provided within.
